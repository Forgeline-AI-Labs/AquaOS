# Getting Started

AquaOS is a Raspberry Pi–powered aquarium controller with a modern web interface.
This page helps you find the right starting point.

## Install on a Raspberry Pi

The [Raspberry Pi Installation Guide](/docs/RASPBERRY_PI_INSTALL.md) walks through
everything from flashing Pi OS to launching the AquaOS dashboard with optional
kiosk mode.

## Build from Source

See [Building from Source](/BUILDING.md) for Go backend and Next.js frontend
build prerequisites and commands.
