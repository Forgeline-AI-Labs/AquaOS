# Architecture

AquaOS is split into two main layers connected by a REST + WebSocket API.

```
┌──────────────────────────────────────────────────┐
│  AquaOS Frontend  (Next.js 14+, App Router)      │
│  TypeScript · Tailwind CSS · Radix UI · Recharts │
└────────────────────┬─────────────────────────────┘
                     │  REST API + WebSocket
┌────────────────────┴─────────────────────────────┐
│  AquaOS Backend  (Go — reef-pi core, extended)   │
│  GPIO · I2C · Outlet scheduling · Dosing · ATO   │
│  Data store: BoltDB (embedded key-value)         │
└────────────────────┬─────────────────────────────┘
                     │
┌────────────────────┴─────────────────────────────┐
│  Raspberry Pi Hardware                           │
│  Pi 4/5 · 7" touchscreen · relay boards · probes │
└──────────────────────────────────────────────────┘
```

## Frontend Stack

| Layer      | Technology                              |
|------------|-----------------------------------------|
| Framework  | Next.js 14+ (App Router)                |
| Language   | TypeScript                              |
| Styling    | Tailwind CSS + CSS custom properties    |
| Components | Radix UI primitives (headless)          |
| Charts     | Recharts                                |
| Real-time  | Native WebSocket                        |
| Icons      | Lucide                                  |

## Backend Stack (inherited from reef-pi)

| Layer      | Technology                              |
|------------|-----------------------------------------|
| Language   | Go                                      |
| API        | REST (net/http) + WebSocket             |
| Data store | BoltDB (embedded)                       |
| Hardware   | GPIO via go-rpio, I2C, PWM              |
| Build      | Make + Go modules                       |

## Design Principles

1. **Touch-first, display-aware** — primary interface is a 7" Pi touchscreen in kiosk mode; every tap target ≥ 44 px.
2. **Themeable from day one** — a design token system (CSS custom properties + Tailwind) powers swappable theme presets.
3. **Wireless by architecture** — the frontend is a standalone web app accessible from any device on the local network.
4. **Keep the backend stable** — reef-pi's Go backend handles hardware abstraction; we preserve its API contract and extend incrementally.
