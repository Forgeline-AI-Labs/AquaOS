# AquaOS Documentation

AquaOS is an open-source, Raspberry Pi–powered aquarium controller. It inherits
[reef-pi's](https://github.com/reef-pi/reef-pi) Go backend for GPIO, I2C, and
peripheral management and pairs it with a modern Next.js frontend built for
touchscreens and mobile browsers.
