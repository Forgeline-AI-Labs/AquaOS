# API Reference

Complete reference for the AquaOS (reef-pi) REST API. All endpoints under `/api/` require
authentication via session cookie (obtained from `POST /auth/signin`). Request and response
bodies are JSON unless noted otherwise.

---

## Authentication

| Method | Path | Request Body | Response | Purpose |
|--------|------|-------------|----------|---------|
| POST | `/auth/signin` | `{ "user": string, "password": string }` | `200` (sets session cookie) or `401` | Sign in |
| GET | `/auth/signout` | ‚Äî | `200` (clears session cookie) | Sign out |
